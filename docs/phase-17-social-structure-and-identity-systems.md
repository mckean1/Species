# Phase 17: Social Structure and Identity Systems

Phase 17 adds the first real social memory layer to polities without changing the canonical monthly flow or replacing the existing material, knowledge, and governance systems.

## What Changed

- Polities now accumulate lightweight social memory tied to lived conditions such as hardship, surplus, settlement continuity, seasonal mobility, coordinated governance, and frontier strain.
- Polities now derive broad identity tendencies from that memory:
  - rootedness
  - mobility
  - communalism
  - autonomy orientation
  - order orientation
  - frontier distinctiveness
- A small set of traditions can now emerge from repeated historical patterns.
- Social identity now feeds back into governance, bloc behavior, and law support/resistance.

## Social Memory

The simulation now tracks aggregate polity memory rather than person-level culture:

- settlement continuity months
- seasonal mobility months
- hardship months
- surplus months
- coordinated governance months
- peripheral strain months
- river-settlement months
- frontier exposure months

These counters accumulate directly from existing settlement, presence, material, pressure, and governance state.

## Traditions

Traditions are intentionally limited and must emerge from recurring conditions:

- `Storage Discipline`
- `Seasonal Return`
- `Frontier Autonomy`
- `Rooted Heartland`
- `Coordinated Order`

They are not assigned randomly. Each one requires sustained simulation history that matches its theme.

## Regional and Internal Distinction

The social layer now lightly distinguishes:

- rooted core / heartland development
- mobile return-cycle identity
- frontier distinctiveness
- autonomy expectations in strained peripheral space

This remains aggregate and polity-owned. There is still no local-culture simulator or detailed social hierarchy layer.

## Governance Feedback

Social identity now affects:

- which laws feel more natural or more strained
- proposal support and opposition
- bloc influence and satisfaction
- compliance with centralized, shared, restrictive, or autonomy-oriented laws
- cohesion and legitimacy under different governance choices

This keeps social character causally tied to existing political systems rather than leaving it as decorative flavor.

## Scope Guardrails

This is still not:

- a full religion system
- a mythology system
- a language simulation
- a class / caste simulation
- a family lineage model
- a diplomacy or warfare overhaul

The current model is intentionally small, aggregate, and meant to support later phases.
